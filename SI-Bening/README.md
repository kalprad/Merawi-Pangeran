# Folder SI-Bening

Folder ini sengaja dikosongkan. File aplikasi SI-Bening (installer/APK)
biasanya berukuran besar, jadi **tidak disimpan di GitHub**. Alurnya:

1. Upload file aplikasi ke Google Drive kalian (di luar GitHub)
2. Salin link share Drive-nya
3. Tempel link itu di halaman **Pengaturan** pada panel admin (`/admin/pengaturan`)
   di kolom "Link Download SI-Bening"

Link itu akan otomatis muncul sebagai tombol unduh di halaman `/si-bening`.
