# Folder Materi Sosialisasi

Folder ini sengaja dikosongkan. File materi (PDF/PPT) biasanya berukuran
besar, jadi **tidak disimpan di GitHub**. Alurnya:

1. Upload file materi ke Google Drive kalian (di luar GitHub)
2. Salin link share Drive-nya
3. Tempel link itu di form **Tambah Materi** pada panel admin (`/admin/materi/new`)
   di kolom "Tautan Berkas"
